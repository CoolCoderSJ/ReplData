API DOCS

To track a repl, just make a GET Request to https://ReplData.coolcodersj.repl.co/api With URL Paramteters ID, TOKEN, and USER. The ID is given on the account page and the TOKEN is also on the account page. The User is a REPL ENV variable called REPL_OWNER. If you are trying to track a web page, just set User to anything you want. Add the following code to the top of your repl's main code. Examples -


Note: Token cannot be in an env since envs do not carry between different users running them unless it is a web repl. It can be hidden though-set up a web server, and make requests to it to access variables from the servers env.

Note that the examples are for NodeJS and Python only, but you can use this for any language that supports a GET request.



Python

	
import requests, os
requests.get(f"https://ReplData.coolcodersj.repl.co/api?ID=REPLDATA_REPL_ID&TOKEN=URTOKEN&USER={os.getenv('REPL_OWNER')}")


NodeJS

	
const fetch = require("node-fetch")
let API = async() = {
	fetch("https://ReplData.coolcodersj.repl.co/api?ID=REPLDATA_REPL_ID&TOKEN=URTOKEN&USER="+process.env.REPL_OWNER)
}
API()
